# <u>OpenWeatherAPI-DataIntegration</u>
This repository includes for building a RealTime Weather Analysis Pipeline.

The weather dataset was collected using OpenWeatherAPI that fetches weather of multiple cities in realtime
The app using ETL collects the data for further analysis
This data is stored in the SQL DB, and then visualized using Streamlit app

This repo contains:
1. SqlDB
2. StreamlitAPP
